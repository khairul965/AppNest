# 🚀 AppNest - Hero IO Assignment

A modern React-based web application where users can explore different apps, view details, and install them easily.

---

## 📌 Live Demo
https://appnst.netlify.app/

---

## 📖 Project Description

AppNest is a responsive web application built using React. It showcases a collection of apps with detailed information such as ratings, downloads, and descriptions.

Users can:
- Browse all apps
- View app details
- Navigate easily between pages
- Experience a clean and responsive UI

---

## ⚙️ Technologies Used

- React.js
- React Router
- Tailwind CSS
- Vite
- JavaScript (ES6)
